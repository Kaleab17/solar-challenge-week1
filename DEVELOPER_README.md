\# Developer Guide



\## Project Overview

This repository is part of the 10 Academy training challenge. It focuses on data cleaning, validation, and analysis of solar datasets from Benin, Sierra Leone, and Togo.



The main goal is to clean and analyze solar radiation data (GHI, DNI, DHI) to find which country has the best solar potential and recommend investment locations for MoonLight Energy Solutions.
